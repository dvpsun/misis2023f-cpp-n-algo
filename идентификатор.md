Идентификатор (имя) 
• начинается с буквы 
• содержит буквы, цифры и подчеркивания
• длинна ограничена реализацией 
• отличается от ключевого слова 
• чувствителен к регистру

Идентификатор (примеры) 
// имя переменной
int32_t **count**(0); 
// имя типа 
**int32_t** count(0); 
// имя функции
int **main**()
